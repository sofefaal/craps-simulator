# 🎲 Craps Simulator

A simple **Craps game simulator** built with **JavaScript** and **React**, fully tested using **Jest** and **React Testing Library**.  

Simulate multiple games, calculate statistics, and view results in a clean React UI.

## Features in the game 

- Roll one or two dice  
- Play a full **Craps game** (win/loss rules, points, 7-out rule)  
- Run **multiple game simulations**  
- Calculate statistics:
  - Wins / Losses
  - Average rolls per game
  - Highest / Lowest rolls
  - Win percentage  
- React UI to run simulations and display results  
- Tested with **unit and component tests**


## Installation 

1. Clone the repository:

```bash
git clone https://github.com/sofefaal/craps-simulator.git
cd craps-simulator
```

2. Install dependencies:
```bash
npm install
```

3. Run the game 

```bash
npm run dev
```

## Testing

To run the tests files, enter in the terminal:

```bash
npm run test
```
